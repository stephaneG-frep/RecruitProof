# RecruitProof

Application Flutter locale pour constituer un dossier volontaire de preuves de
recherche d’emploi.

## Principes de confidentialité

- aucune création de compte ;
- aucun serveur ni service Firebase ;
- aucune surveillance des applications ou de la navigation ;
- seules les informations et preuves choisies par l’utilisateur sont stockées ;
- les données restent dans le stockage local de l’application.

## Lancer le projet

```bash
flutter pub get
flutter run
```

Plateformes générées : Android, Linux et Web.
